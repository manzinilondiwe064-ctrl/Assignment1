# Assignment1
This is an app that shows different times of the day (Morning, Afternoon and Evening)
This app has been tested and run with an emulator 
After the app runs the user can input anytime of the day they want output of
If the user input MORNING the output/display will be send a "Good morning" text to a family member.
If the user input Mid-morning the display will be Reach out to a colleague with a quick "Thank you"
If the user input Afternoon display will be Share a funny meme or insteresting link with a friend
If the user input Night display will be Leave a thoughful comment on a friend"s post
